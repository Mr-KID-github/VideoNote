# 介绍

VINote 是一个把视频转成结构化 Markdown 笔记的工作台，围绕这条主流程设计：

1. 登录
2. 提交视频 URL
3. 后端下载、转写、总结
4. 轮询任务直到完成
5. 保存并继续编辑 Markdown 笔记

## 这套文档负责什么

这套文档负责“人类可读的使用说明”：

- 产品工作流
- 认证模型
- API 路由分组
- 集成说明
- 本地部署与局域网使用

## 这套文档不重复什么

FastAPI 已经提供 OpenAPI 参考文档，所以不应该把字段表再手写一遍。

- Swagger UI: `http://127.0.0.1:8900/docs`
- ReDoc: `http://127.0.0.1:8900/redoc`

如果这里的说明和 Swagger 不一致，以 Swagger 为准，然后再回来修正文档。
