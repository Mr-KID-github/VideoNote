# Quickstart

## Run the app

Backend:

```bash
pip install -r requirements.txt
python main.py
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Docs:

```bash
cd docs
npm install
npm run docs:dev
```

Default local addresses:

- App: `http://localhost:3100`
- Backend API: `http://127.0.0.1:8900`
- Swagger: `http://127.0.0.1:8900/docs`
- Docs site: `http://localhost:3101`
